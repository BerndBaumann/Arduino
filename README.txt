### Sensor things ###

Implementation of a custom communication interface using light signals.
The transmitter uses a LED for sending, the receiver uses a photo transistor
as a sensor.

The file Doku.pdf contains a full documentation of the used methods and
hardware.

### Roboter with 4 Servos ###

Program to control a soccer playing, walking robot based on a ATmega328p

### for the fun ###

Program to simulate a dice game.
